# The E-shop cat Project

Vous allez découvrir notre projet

## Merci de tester le projet en local

Afin de tester notre code : téléchargez le projet.
Ensuite, suivez la liste d'instructions : 
```
bundle install
```

```
rails db:create
rails db:migrate
rails server
```
